# Aurora Scheduler

A TypeScript program that helps solar installers prioritize and manage their installation crew. Given a list of buildings ordered by priority and a list of employees with per-day availability, it generates a 5-day work schedule (Monday–Friday) showing which employees are assigned to which building each day.

## Install

```bash
npm install
```

## Run

```bash
npx ts-node src/index.ts
```

## Test

```bash
npx jest
```
